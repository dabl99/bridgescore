# Bridge Score Server

A bridge table scoring database written using [Express 4](http://expressjs.com/).

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) and the [Heroku Toolbelt](https://toolbelt.heroku.com/) installed.

```sh
$ git clone git@github.com:bradphelan/bridgescore.git or clone your own fork
$ cd bridgescore
$ npm install
$ npm start -p 7000
```

Your app should now be running on [localhost:7000](http://localhost:7000/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku master
$ heroku open
```
or

[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy)
